# Team Agency - JavaScript Capstone Project

Team Agency is a modern JavaScript capstone project built using pure HTML, CSS, and JavaScript. It brings multiple interactive web applications together inside one premium dashboard-style interface.

## Live Project

https://advanced-java-script-project.vercel.app

## Project Overview

This project is designed as a complete frontend capstone where different JavaScript-based applications are connected through a single responsive interface. The main portfolio page introduces the team, displays project highlights, and provides smooth navigation between all apps.

## Features

* Premium glassmorphism sidebar
* Interactive project navigation
* Fully responsive layout from 320px+
* Modern dark UI design
* Smooth hover effects and animations
* Single-page project switching
* Built using HTML, CSS, and JavaScript only

## Purpose of the Project

The purpose of this project is to practice and demonstrate advanced JavaScript concepts through real-world mini projects. Each section focuses on DOM manipulation, events, localStorage, API handling, form validation, drag and drop, and dynamic UI updates.

## Team Agency

Team Agency represents a collaborative capstone project where multiple JavaScript applications are combined into one clean and professional web experience.
